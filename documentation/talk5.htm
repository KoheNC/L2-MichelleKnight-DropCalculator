GM Talkthrough - Chat and Logs

Server Console

The server console is, on initial entry, a list of the last fifty or so lines of the console messages. It also refreshes itself every five seconds and, after the first refresh, it will automatically position itself at the bottom of the list, so you can see the most recent messages. This is the format of all the chat viewing systems. There is also a button at the top left of the screen, which will display the whole file, rather than just the last few lines, but once this option is selected, the automatic refresh stops for obvious reasons. Pushing that button again will return to the last few lines list and the refresh will resume.

Chat Log

The Chat Log simply displays the chat that is going on in the system. Like the server console, the initial entry brings up the last fifty or so lines of the chat recorded by the server, and it refreshes every five seconds. Some versions of the drop calc also have a button in the bottom right, for GM's that have the announce ability to make server announcements.

Login Console - Login Events

These two options simply enable you to see the current output of the logon servers console and events files. It operates in the same way as the server console and chat log options.
