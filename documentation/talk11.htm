GM Talkthrough - Shops and Pets

The shops, although not creatable by the drop calc, are certainly editable by it.

This section makes changes to the database shop entries, not the live shops, so the server has to be restarted before any changes made here, are seen. The list of shops is generated automatically acording to whatever shops it finds in the database, which is why the columns can sometimes look a bit out of length. The list is shown according to shop ID, rather than alphabetically.

 

On entering this section, the contents of the shop is split over two columns. A box at the top allows a new item to be added to the ending list of the shop, while buttons by the side of each item name allows you to change its price, delete it, or the arrows allow you to change the order of the items.

 

Here, you can do a little pet maintenance. The list of pets will show you all the pets on the server, listed by the level of the pet. If a pet is found that doesn't have an owner, there is a clickable link that will allow you to, first, specify the type of the pet (as the type is determined by the type of caller used to call it, and if that is missing, then the race of the pet can not be known) and then a list of all the characters is generated, for you to select a character to give the caller to.

The new caller will appear in the characters inventory, but if the character is online at the time, then they will need to re-log before they see the caller. It has to be given through a database entry, as extra reference details need to be recorded that can not be given through the telnet give command.

It is also possible for the admin to clear the pets name. Usually, naming is a one-shot deal for the user, but by clearing the name the user is able to give the pet another name.
